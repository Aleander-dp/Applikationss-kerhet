# Applikationss-kerhet
Applikationssäkerhet och test av OWASP Juice Shop

## filer
Rapport av säkerhetsbrister i OWASP Juice Shop

## Syfte 
Syftet med detta projekt var att få en förståelse för hur applikationer säkerhetstestas genom att använda både manuella medel och automatiserade säkerhetsverktyg. I projektet kartlaggde vi applikationens attackytor, hittade sårbarheter av lägre och högre grad, bedömde påverkan av dessa med CIA-triaden, Dokumenterade besvis och gav rekommenderade åtgärder till varje sårbarhet.

## Använda program
- Docker (isolerad testmiljö)
- Nuclei (automatiserad scanning)
- FFUF (endpoint discovery)
- BurpSuite (trafikmanipulation)
- SQLmap (SQL-injektionstester)
- Crackstation (hash-cracking)
- OWASP Juice Shop (applikation)
